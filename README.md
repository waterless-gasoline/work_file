# 彭煌城个人项目展示网站

基于 Next.js 16 + Tailwind CSS 搭建的个人作品集网站，用于展示 NLP、RAG、多模态、端侧部署与模型优化相关项目。

## 本地运行

```bash
npm install
npm run dev
```

打开 `http://localhost:3000` 查看。

## 当前页面

- `/` 首页
- `/projects` 项目列表页
- `/projects/[slug]` 项目详情页

## 简历

简历文件放在 `public/resume.pdf`，首页可直接下载。
